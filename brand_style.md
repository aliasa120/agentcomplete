# THE ECHO — Brand Style Guide for Image Editing

## Brand Identity

| Element | Value |
|---|---|
| Brand name | **THE ECHO** |
| Watermark | `theecho.news.tv` |
| Primary Color | Deep Teal `#1A5C5A` |
| Highlight Color | Warm Mustard `#C9A227` |
| Text Color | White `#FFFFFF` |
| Dark Card | `#0D1F1E` |
| Brand mark | TOP-LEFT corner — "THE ECHO" wordmark on small solid teal `#1A5C5A` badge |
| Watermark position | BOTTOM-RIGHT corner — "theecho.news.tv" in small mustard `#C9A227` text |

---

## Style Selection — Pick 1 Based on News Type

| News Type | Style Number | Style Name | Visual Feel |
|---|---|---|---|
| Breaking / Hard News / Political crisis | **Style 1** | Gritty Ground-Level | Real photo + deep teal gradient from bottom 30%, white bold serif headline, mustard kicker label |
| Quote / Interview / Controversy / Person-focused | **Style 2** | Portrait Gradient | Close-up face dominant, dark gradient overlay, bold sans headline in white, mustard `#C9A227` word highlights |
| Feature / Human Interest / Nature / Science / Society | **Style 3** | Clean Container | Wide cinematic shot, white/teal rounded rectangle text block at bottom, clean minimal layout |
| Tabloid / Geo-political Drama / Scandal / Opinion | **Style 4** | Composite Dramatic | Bold all-caps headline, yellow/red word highlights, distressed textures, high contrast |
| Tech / Economy / Finance / Environment / Niche | **Style 5** | Cinematic Branded | High-angle or wide shot, solid black text block at bottom 28%, single teal `#1A5C5A` vertical accent bar on left edge |
| Disaster / Grief / Humanitarian crisis / War | **Style 6** | Immersive Dark Band | Solid dark teal `#1A5C5A` bands top 15% + bottom 22%, white serif centered headline, sombre/respectful tone |

---

## 3 Required Text Layers (Every Image Must Have All 3)

| Layer | Length | Position | Color | Font Style | Purpose |
|---|---|---|---|---|---|
| **KICKER** | 2–4 words | Above headline | Mustard `#C9A227` | Small caps, sans-serif | Category label |
| **HEADLINE** | Max 10 words | Center of overlay | White `#FFFFFF` | Bold, serif-feel | Hook from X post |
| **SPICE LINE** | Max 15 words | Below headline | White `#FFFFFF` | Smaller, italic | Intriguing teaser — NOT a repeat of headline |

### Kicker Examples by News Type
- Breaking news: `BREAKING NEWS`
- Court/legal: `COURT RULING`
- Financial: `MARKET UPDATE`
- Exclusive: `EXCLUSIVE`
- Diplomatic: `DIPLOMATIC CRISIS`
- Political: `POLITICAL STORM`
- Tragedy: `TRAGEDY`
- Interview: `EXCLUSIVE INTERVIEW`

---

## Mandatory Preservation Rule

**Every editing prompt MUST end with this exact sentence:**
> Preserve original photo quality, sharpness and colors exactly — only add overlay and text. Do not upscale, blur, or re-compress.
